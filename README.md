Repo for maintaining icp daily deployment on KVM PowerVM

## Load the jenkins jobs

```shell
# make load-jenkins-jobs JENKINS_USER=<jenkins user> JENKINS_PASSWORD=<jenkins password> JENKINS_URI=<>

```
